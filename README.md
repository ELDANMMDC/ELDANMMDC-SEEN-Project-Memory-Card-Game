## Overview
SEEN is a memory card game designed to sharpen recall, improve concentration, and enhance problem-solving skills. The game emphasizes minimalism, clarity, and responsiveness, ensuring users can engage with the gameplay without distractions across different devices.  

The design follows a user-centered approach guided by Human–Computer Interaction (HCI) principles. Every element prioritizes accessibility, cognitive support, and efficient gameplay.  

## Features
- Minimalistic interface with limited elements  
- Responsive design across devices  
- Landing Page with clear “PLAY NOW” and “PLAY FOR FREE” options  
- Sequential main page flow (4 clicks to gameplay)  
- Gameplay screen showing only score, moves, misses, accuracy, reset button, and cancel button  
- Post-game summary with performance stats and replay options  
- Future features: Multiplayer, Progress Tracking, Reward System

## Gameplay
1. On the landing page, click the **PLAY** button to start the game  
2. Sequential setup flow:  
   - Pick the **board size**  
   - Choose a **theme**  
   - Select the **game speed**  
3. Match cards to test your memory  
4. After the game ends, view a **post-game summary** showing performance stats and replay options
